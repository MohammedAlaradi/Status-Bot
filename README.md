# How To Use

Status Bot is developed for personal use, so expect some inconveniences. However, the source code is simple and has comments, so don't hesitate to edit it yourself.

You need python-dotenv library or simply hard-code the token, server ID and channel ID, though I don't recommend it, it is severe security hazard.

<table>
<tr> <th>Variable</th> <th>Description</th> <th>Data Type</th> </tr>
<tr> <td>DISCORD_TOKEN</td> <td>Bot token</td> <td>String</td> </tr>
<tr> <td>SERVER_ID</td> <td>The server you want to bot operate</td> <td>integer</td> </tr>
<tr> <td>CHANNEL_ID</td> <td>The chat you want the bot to send the log before resetting</td> <td>integer</td> </tr>
</table>
